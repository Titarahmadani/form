Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan Ujicoba

  • Tombol "Reset" berfungsi untuk menghapus data yang telah dimasukkan di kolom input teks.
  • Tombol "Kirim" berfungsi untuk mengirim data yang telah dimasukkan di kolom input teks ke halaman tujuan.
  • Atribut `target` pada tag menentukan di mana halaman tujuan akan dibuka.
  • Atribut `method` pada tag menentukan cara data formulir akan dikirim ke halaman tujuan.
